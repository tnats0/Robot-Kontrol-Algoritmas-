AtayBot Kontrol,Navigasyon ve Güdüm Algoritmaları
=======

Şu anda AltayBot'un yazılımı için tasarlanmış olan algoritmalarının koda aktaralışının erken aşamasındaki çalışmaya göz atıyorsunuz. Algoritmalar elektronik devreler için talep edilen malzemeler ulaştığı vakit elektornik tasarım süreciyle beraber tamamlanacaktır.
